# \# Soluzione Esercizio Modulo 4 - Corso Coursera

# 

# Questo repository contiene la soluzione per l'incarico finale del modulo 4 del corso \*\*"HTML, CSS, and Javascript for Web Developers"\*\* della Johns Hopkins University.

# 

# \## 📝 Descrizione

# L'obiettivo dell'esercizio è iterare su un array di nomi e stampare un saluto diverso nella console del browser in base alla lettera iniziale del nome:

# \* Se il nome inizia con \*\*J\*\* o \*\*j\*\*, stampa: `Good Bye \[Nome]`

# \* Se inizia con qualsiasi altra lettera, stampa: `Hello \[Nome]`

# 

# \## 🚀 Come visualizzare il progetto

# Per vedere il risultato dell'esercizio:

# 1\. Visita la pagina live qui: \[https://alesalzillo2000-maker.github.io/M4\_solution/](https://alesalzillo2000-maker.github.io/M4\_solution/)

# 2\. Apri gli \*\*Strumenti per sviluppatori\*\* del browser (tasto `F12` o tasto destro -> `Ispeziona`).

# 3\. Clicca sulla scheda \*\*Console\*\*.

# 

# \## 📁 Struttura dei file

# \* `index.html`: Pagina principale che carica gli script.

# \* `script.js`: Logica principale del ciclo e del controllo dei nomi.

# \* `SpeakHello.js`: Modulo per il saluto "Hello".

# \* `SpeakGoodBye.js`: Modulo per il saluto "Good Bye".
